Share quotes via twitter

This is a proof of concept to make it more comfortable to share selected text via twitter. To start, simple select the text you want to tweet. Currently works in the latest versions of IE, Opera, Firefox and Chrome.

If you use an old browser or JavaScript is not available the tweet button at the bottom is still usable.

Once someone clicks the url in the tweet, the script on the site highlights your quote and scrolls into view.

Your feedback is highly appreciated and will help me to improve this script. Please contact me on twitter @justmarkup
